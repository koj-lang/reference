# Function Types

> **<sup>Syntax</sup>**\
> _FunctionType_ :\
> &nbsp;&nbsp;&nbsp;&nbsp; `(` ( [_Type_](/type_system/index.md) ( `,` [_Type_](/type_system/index.md) )<sup>*</sup> )<sup>?</sup> `)` `->` [_Type_](/type_system/index.md) 

Example:

A function adding two integers together may have the following type:

```
(int, int) -> int
```
